# 🐍 Python Beginner Projects

A collection of small Python projects built during my first month of learning programming. Each project helped me practice core concepts like logic, loops, functions, and user input.

> 👉 Looking for more advanced work? Check out my [Python Projects repo](https://github.com/your-username/python-projects)!

---

## 📁 Projects

### 👋 Hello World
> The classic first program — the beginning of every programmer's journey.

**Concepts practiced:** `print()`, getting started with Python

---

### 🎸 Band Name Generator
> Enter your hometown and pet's name to generate a fun band name.

**Concepts practiced:** `input()`, string concatenation, variables

---

### 💰 Tip Calculator
> Calculates how much each person should pay after splitting a bill with tip.

**Concepts practiced:** `float`/`int` type conversion, arithmetic operators, `round()`


---

### 🗺️ Treasure Island
> A text-based choose-your-own-adventure game. Make the right choices to find the treasure!

**Concepts practiced:** nested `if/elif/else` conditionals, user input, string methods


---

### ✂️ Rock Paper Scissors
> Play Rock Paper Scissors against the computer with ASCII art visuals.

**Concepts practiced:** `random` module, lists, conditionals, `input()`


---

### 🔐 Random Password Generator
> Generates a randomized, shuffled password based on your chosen number of letters, symbols, and numbers.

**Concepts practiced:** `random.choice()`, `random.shuffle()`, loops, lists


---

### 🔢 Number Guessing Game
> The computer picks a random number — try to guess it within a limited number of attempts across easy, medium, and hard difficulties.

**Concepts practiced:** `random.randint()`, `while` loops, `if/elif/else`, difficulty levels with dictionaries


---

### 🔤 Hangman
> Guess the hidden word one letter at a time before the hangman is complete.

**Concepts practiced:** `random`, functions, lists, loops, ASCII art stages


---

### 🔑 Caesar Cipher
> Encrypt or decrypt a message by shifting letters along the alphabet.

**Concepts practiced:** string manipulation, modulo operator `%`, functions with parameters, loops


---

### 🧮 Calculator
> A terminal calculator that supports chaining operations using the previous result.

**Concepts practiced:** functions, dictionaries of functions, `while` loops, type conversion


---

### 📈 Higher Lower Game
> Guess which celebrity has the higher net worth — keep your streak going as long as you can!

**Concepts practiced:** `random`, dictionaries, functions, game loops, f-strings


---

### 🔨 Secret Auction
> A blind auction where multiple bidders enter their bids privately, and the highest bidder wins.

**Concepts practiced:** dictionaries, loops, `input()`, finding max values


---

### 🃏 Blackjack
> Play a full game of Blackjack against the computer, complete with ASCII card art and proper scoring logic.

**Concepts practiced:** functions, `random`, lists, game logic, score calculation, `while` loops


---
